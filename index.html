Head

The Head Tags Purpose The <head> tag is where you manage things like the title of the page and what to connect to and import. for example fonts, icons, styling pages and etc...

  • Step 1: Start by adding the HTML tag head inside of your html tag.
    Syntax <html>
    <head> ... </head>
    </html>
  • Step 2: Add a <meta />Meta tags are HTML elements that provide information about a web page's content to search engines and browsers, influencing its indexing, display, and search ranking
    Syntax <head>
    <meta />
    </head>
    • Charsets - Charsets in HTML specify the character encoding used to interpret and display text, ensuring proper rendering of international characters and symbols on web pages.
      UTF-8 is the most widely used character encoding for the web. It can represent every character in the Unicode character set Syntax <meta charset="UTF-8" />
      Also known as Latin-1, this encoding covers Western European languages, including characters used in English, French, German, and Spanish. Syntax <meta charset="ISO-8859-1" />
      An encoding capable of representing all Unicode characters. It exists in two forms: Big Endian (UTF-16BE) and Little Endian (UTF-16LE). Syntax <meta charset="UTF-16" />
      A character encoding used by Microsoft Windows for Western European languages. It is a superset of ISO-8859-1 with additional characters. Syntax <meta charset="Windows-1252" />
      Also known as Latin-2, this encoding supports Central and Eastern European languages, such as Polish, Czech, and Hungarian. Syntax <meta charset="ISO-8859-2" />
      Similar to ISO-8859-1 but includes additional characters, such as the euro sign (€) and other characters missing from ISO-8859-1. Syntax <meta charset="ISO-8859-15" />
      A character encoding for Japanese, which includes a mix of single-byte and double-byte characters. Syntax <meta charset="Shift_JIS" />
  • Step 3: Add a UTF-8 meta tag inside of your <head>
    Syntax <head>
    <meta charset="UTF-8" />
    </head>
  • Step 4: Add a title tag inside of your <head>
    Syntax <head>
    <meta charset="UTF-8" />
    <title>My Awesome Site</title>
    </head>

Congrats, you just made your head, for your page !!!
